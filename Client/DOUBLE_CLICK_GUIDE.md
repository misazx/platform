# 🎮 双击启动使用指南

## ✅ 已创建的启动方式

你现在有 **2 种双击启动** 方式：

---

### 方式一：macOS 应用程序（推荐）🌟

**文件位置**: `打包工具.app`

**特点**:
- ✅ 真正的应用程序，像其他 Mac 应用一样
- ✅ 有自定义图标
- ✅ 可以固定到 Dock 栏
- ✅ 可以添加到 Launchpad

**使用方法**:

1. **直接双击** `打包工具.app` 文件即可启动！

2. **（可选）固定到 Dock**：
   - 启动应用后，右键点击 Dock 中的图标
   - 选择 **"选项" → "在 Dock 中保留"**

3. **（可选）移动到应用程序文件夹**：
   ```bash
   # 复制到 Applications 文件夹（这样可以在 Launchpad 中找到）
   cp -r "打包工具.app" /Applications/
   ```

---

### 方式二：.command 脚本（备选）

**文件位置**: `启动打包工具.command`

**特点**:
- ✅ 双击即可运行
- ✅ 会打开终端窗口显示日志
- ⚠️ 首次运行需要授权

**使用方法**:

1. **首次运行**：
   - 双击 `启动打包工具.command`
   - 如果弹出提示，点击 **"打开"**
   
2. **如果提示无法执行**：
   ```bash
   # 在终端中执行一次：
   chmod +x "启动打包工具.command"
   ```
   然后再次双击即可

## 🔧 故障排除

### 问题 1：双击 .app 没反应

**解决方案**：

```bash
# 方法 A: 通过终端启动（查看错误信息）
open "打包工具.app"

# 方法 B: 直接运行可执行文件
"打包工具.app/Contents/MacOS/打包工具"
```

如果看到错误信息，根据提示解决。

---

### 问题 2：提示"无法验证开发者"

**原因**: macOS 安全机制阻止了未签名的应用

**解决方案**：

1. **临时允许**：
   - 打开 **系统设置 → 隐私与安全性**
   - 在"安全性"部分，你会看到关于"打包工具"的消息
   - 点击 **"仍要打开"**

2. **永久允许**（推荐）：
   ```bash
   # 移除隔离属性
   xattr -cr "打包工具.app"
   
   # 再次双击即可
   ```

---

### 问题 3：提示 Python 或 tkinter 错误

**解决方案**：

```bash
# 安装 Python 和 tkinter
brew install python@3.11 python-tk@3.11

# 如果使用系统自带 Python（无需安装 tkinter）
/usr/bin/python3 build_gui.py
```

## 💡 高级技巧

### 添加到 Dock 固定位置

1. 启动 `打包工具.app`
2. 右键点击 Dock 中的图标
3. 选择 **选项 → 在 Dock 中保留**

以后就可以一键从 Dock 启动！

### 创建桌面快捷方式

```bash
# 创建桌面别名
ln -s "$(pwd)/打包工具.app" ~/Desktop/打包工具.app
```

### 从 Launchpad 启动

```bash
# 复制到 Applications 文件夹
sudo cp -r "打包工具.app" /Applications/

# 然后可以在 Launchpad 中找到并启动
```
