# Inversion Exercise

## Overview

When you're stuck because "it must be done this way", invert the problem.

**Core principle:** Instead of asking "How do I achieve X?", ask "How do I guarantee X fails?"

## The Process

1. **State the goal** clearly
2. **Invert it** - "How would I make this fail?"
3. **List all failure modes** - Every way to guarantee failure
4. **Invert back** - The opposite of each failure mode is a requirement
5. **Check assumptions** - Which "requirements" are actually assumptions?
