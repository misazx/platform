# Meta-Pattern Recognition

## Overview

When the same issue keeps appearing in different places, there's a deeper pattern to find.

**Core principle:** Look for the meta-pattern that connects seemingly different problems.

## The Process

1. **List the instances** - Where have you seen this before?
2. **Abstract the pattern** - What's common across all instances?
3. **Name the pattern** - Give it a clear, memorable name
4. **Find existing solutions** - Has this pattern been solved elsewhere?
5. **Apply universally** - Fix all instances with one approach
